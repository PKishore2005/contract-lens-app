### Prerequisites
- Node.js 18+

### Setup

1. **Clone the repository**
```bash
git clone https://github.com/YOUR_USERNAME/contract-lens.git
cd contract-lens
```

2. **Install dependencies**
```bash
npm install
```

3. **Environment Setup**
```bash
cp .env.example .env
# Add your Google Gemini API key to .env
```

4. **Start development server**
```bash
npm run dev
```

5. **Open in browser**
```
http://localhost:5173
```
